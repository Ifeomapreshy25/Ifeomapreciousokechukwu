Tableau

With Tableau, I can connect to various data sources, analyze trends (e.g., sales performance, customer behavior), and present clear insights that support informed business decisions..

Power Bi

Power BI is a business analytics tool I use to create interactive dashboards and reports that visualize data and uncover insights. It allows me to connect to multiple data sources, perform data modeling, and present clear, actionable findings to support decision-making.

R-programming

R Programming is a statistical computing language I use for data analysis, visualization, and modeling. With R, I can clean and manipulate data, perform advanced statistical analyses, and create high-quality visualizations to support data-driven insights.

Python

Python is a versatile programming language I use for data analysis, automation, and building data-driven solutions. With its powerful libraries like Pandas, NumPy, and Matplotlib, I can clean, analyze, and visualize data efficiently to extract meaningful insights.

Excel

Microsoft Excel is a powerful tool I use for data organization, analysis, and visualization. With functions, formulas, pivot tables, and charts, I efficiently clean, analyze, and present data to support informed decision-making.

SQL

SQL (Structured Query Language) is a powerful tool I use to query, manage, and analyze data stored in relational databases. With SQL, I can extract relevant data, perform joins, filter large datasets, and generate reports to support data-driven decisions.
